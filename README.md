# credit-risk-modelling-UK
IFRS 9 modelling (PD, EAD, LGD) with UK regulations, NPV &amp; Scorecard modelling and macro-economic stress testing

A complete, end-to-end credit risk modeling project built using real-world loan data (Home Credit Default Risk), focused on:

Predicting Probability of Default (PD)
Simulating IFRS9 staging and ECL
Delivering transparency with SHAP (XAI)
Aligning with regulatory and business expectations in UK challenger banks

🧱 Project Structure
Phase	Deliverable
1. Data Cleaning	Missing value handling, logical imputations
2. Feature Engineering	Age, income ratios, employment stability, binary flags
3. EDA	Class imbalance, risk drivers, visual insights
4. Baseline Modeling	Logistic Regression (PD), ROC-AUC evaluation
5. Explainability	SHAP summary plots, individual decision analysis
6. Credit Policy Logic	Cutoffs for Stage 1/2/3 + ECL estimates
7. (Optional)	Streamlit interface for live scoring
